Tema 1 - Propiedades de Texto en CSS

Propiedad Valor Inicial Descripción
color depends on user agent Define el color del texto.
font-family depends on user agent Especifica la familia tipográfica del texto.
font-size medium Determina el tamaño del texto.
font-weight normal Establece el grosor del texto (negrita, normal, etc.).
font-style normal Define el estilo del texto (normal, itálico, etc.).
font-variant normal Configura variantes del texto como small-caps.
font initial Es una abreviatura que permite definir todas las propiedades de la fuente a la vez.
text-align initial Especifica la alineación del texto.
line-height normal Define la altura de la línea del texto.
text-decoration none Establece decoraciones como subrayado, tachado, etc.
text-transform none Controla la capitalización del texto (mayúsculas, minúsculas, etc.).
vertical-align baseline Alinea verticalmente el texto en relación con su línea base.
text-indent 0 Define la sangría del texto en la primera línea de un párrafo.
letter-spacing normal Establece el espacio entre caracteres.
word-spacing normal Define el espacio entre palabras.
white-space normal Controla cómo se manejan los espacios en blanco dentro del texto.

Tema 2 - Características de Enlaces en CSS

Característica Propiedades Descripción
Tamaño, Color y Decoración
  • font-size
  • color
  • text-decoration
 Controla el tamaño del texto del enlace, el color del enlace y la decoración (por ejemplo, subrayado).
Decoración Personalizada
  • text-decoration
  • text-decoration-color
  • text-decoration-style
  • text-decoration-thickness
 Permite personalizar la decoración del enlace, como cambiar el color del subrayado, el estilo de la línea (sólida, punteada, ondulada, etc.) y el grosor de la decoración.

Tema 3 - Características de Imágenes en CSS

Característica Propiedades Descripción
Anchura y Altura
  • width
  • height
 Define la anchura y la altura de la imagen. Puede establecerse en valores fijos (por ejemplo, píxeles) o relativos (por ejemplo, porcentajes).
Eliminar el Borde
  • border
  • border-style
  • border-width
 Permite eliminar el borde que aparece alrededor de una imagen, especialmente cuando es un enlace. Esto se logra configurando la propiedad border a none.

Tema 4 - Listas

Propiedad Valor Inicial Descripción
list-style-type disc Define el tipo de marcador (disco, círculo, cuadrado, número, etc.) para los elementos de la lista.
list-style-position outside Establece la posición del marcador respecto al contenido del elemento de la lista (inside o outside).
list-style-image none Permite utilizar una imagen como marcador en lugar de los tipos de marcador predeterminados.
list-style Combina list-style-type, list-style-position y list-style-image Es una propiedad abreviada que define todos los aspectos del estilo de lista en una sola declaración.

Tema 5 - Tablas

Propiedad Se Aplica a Valor Inicial Descripción
border-collapse Elementos de tabla (table) separate Define si las celdas adyacentes comparten un borde común o si tienen bordes separados.
border-spacing Elementos de tabla (table) 0 Establece la distancia entre los bordes de celdas adyacentes cuando border-collapse está en separate.
empty-cells Celdas de tabla (td, th) show Determina si se muestran o se ocultan las celdas vacías de la tabla.
caption-side Elemento caption de tabla top Establece la posición de la leyenda de la tabla, ya sea en la parte superior o inferior de la misma.
